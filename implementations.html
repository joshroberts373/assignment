Inline css

Inline css is used to apply a unique style to the html element. To use the inline css element you will need to use a style attribute.

Internal css

Internal css is used to define a style for your html page. To use a internal element you would need to use a head element within the style element.

External css

External css sheet is used so that the style is difined for different html pages. In order to use a external style sheet you need to add a reference/link to that page within the head section of each page used.
