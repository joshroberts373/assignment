Box Model

The CSS box model is a box that wraps around every HTML element. It consists of borders, padding, margins, and the content.

The border goes around the padding and the content. The content is where the developer has put the contents of the page in. The padding clears the area around the content. The margin clears the area outside the border. 

    width: 300px;
  padding: 15px;
  border: 15px solid green;
  margin: 20px;
