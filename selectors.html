CSS slectors

CSS selectors are used to find or select the html element style that you need. You can put CSS sselectors into five different catogories: simple, combinator, pseudo-class, pseudo-elements and attribute.

Universial Seader (*)

If you use (*) universal selector it will allign and put all text in blue that is within the body.

Element Selector

The element selector will select all the elements that has that specific nameed element attached to it.

Class Selector

To use the class selector you will use .class, to then make a class you will use the period (.) then the name of the class.
